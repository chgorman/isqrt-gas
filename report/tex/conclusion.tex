\section{Conclusion}

After comparing various algorithms for total gas cost,
we found that the most efficient algorithm based on the sampled values is
\UnrolledThree{} (Listing~\ref{list:newton-unrolled-3}).
A proof of correctness may be found Appendix~\ref{app:proofs}.

In Appendix~\ref{app_asymptotics:error_newton},
we see that the different initializations change
the required number of iterations but not asymptotically;
that is, using Newton iteration still requires $O(\lg \lg n)$ iterations
to compute the integer square root%
\footnote{Thus, improved initializations give \emph{practical}
but not \emph{theoretical} improvements.}.
In Appendix~\ref{app:isqrt_with_float_sqrt},
we work through the details of algorithms which use
\emph{one} floating-point square root as part of the computation
(as suggested by~\cite[Chapter 1.7, Remarks]{cohen1993}).
It should be noted that any \emph{significant} computation
should use a dedicated library for these computations
such as \texttt{GMP}~\cite{gmplib} or \texttt{Flint}~\cite{flint}.
